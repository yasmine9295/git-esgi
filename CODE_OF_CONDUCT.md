# Code de Conduite

Pour que le projet reste un espace agréable et respectueux, tous les participants doivent suivre ces règles :

1. Les participants doivent être tolérants face aux opinions différentes.
2. Les participants doivent s'assurer que leur langage et leurs actions ne contiennent aucune attaque personnelle ni remarque désobligeante.
3. Lors de l'interprétation des paroles ou actions des autres, les participants doivent toujours partir du principe que les intentions sont bonnes.
4. Tout comportement pouvant être raisonnablement considéré comme du harcèlement ne sera pas toléré.
